Thai Reading Addon

Visual reference for all display and reading options

Reading Types

RTGS (Royal Thai General System)

sà-wàt-dee khráp

IPA (International Phonetic Alphabet)

sà.wàt.diː kʰráp

Display Types

Each active-field entry chooses one display type. Hover cards show hover behaviour.

Thai Tones

Thai has 5 phonemic tones. Each is configurable via ThaiTones in config.

Configuration Reference

KeyTypeDefaultDescription
